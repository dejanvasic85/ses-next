This plan covers the phased SEO and GEO (Generative Engine Optimisation) improvements for sesmelbourne.com.au. Work is structured as monthly sprints aligned to the $140/month subscription, with any additional work communicated and approved upfront.

| Item                                          | Cost              |

| Hosting + framework updates (Next.js, Sanity) | $40/month         |
